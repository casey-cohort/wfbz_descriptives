# WFBZ Descriptives

See: https://casey-cohort.github.io/wfbz_descriptives/

## Dependencies

R, Quarto, Conda, and the following R packages: 

* tidyverse
* sf
* patchwork
* gt
* glue
* showtext
* tidycensus
* ggh4x
* tigris

For the python environment, run: 

```
conda create -n wfbz numpy pandas geopandas jupyter
conda activate wfbz
pip install popexposure
```


## Data Prep

Manually download data set from 
https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/DWILBW 
and unzip to data/raw. 

## Rendering

To update this page on Github Pages, run:

```
quarto publish gh-pages --no-prompt
```

To render locally, use:

```
quarto preview
```

## What's What? 

* `assets/` contains font files
* `data/` populated by running `prep_data.R` and contains preprocessed data sets 
* `figures/` individual figure image files
* `rendered/` website files (don't touch... generated by quarto)
* `reports/` quarto documents for producing webpage and figures
